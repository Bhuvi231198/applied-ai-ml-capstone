# Applied AI & ML Essentials Capstone Project

This repository contains all four parts of the Applied AI & ML Essentials Capstone Project.

## Parts

- Part 1 - Data Acquisition, Cleaning, and Exploratory Analysis
- Part 2 - Supervised Machine Learning Model — Build, Train, and Evaluat
- Part 3 - Advanced Modeling — Ensembles, Tuning, and Full ML Pipeline
- Part 4 - LLM-Powered Feature: Structured Extraction, Tabular Batch Scoring, or Model Prediction Explanation

Each part has its own README.md explaining the implementation, dependencies, and execution steps.
